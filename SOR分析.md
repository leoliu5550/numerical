# Conclusion
1. 由迭代次數觀察，解答該線性方程組所需要的步數(N_iter)，和 Matrix Size 無關
2. 由迭代次數觀察，解答該線性方程組所需要的步數(N_iter)，落在0.9 和 1.0,表示這個問題適合用GS Method來解，需要的步伐數最少。
